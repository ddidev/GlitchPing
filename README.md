# GlitchPing
NodeJS Application for pinging glitch projects to keep them online 24/7

# How To Use
1. Install the following
- request
- http
- nodejs

2. Set the glitch links (and add more if you wish)

3. Start it using your preffered option:

PM2:
pm2 start ping.js

FOREVER:
forever ping.js
